Love of big letters

Write a program that reads a string and replaces all even characters with uppercase letters and all odd characters with lowercase letters (index 0 is considered even). Print the result.

Use StringBuilder to solve the problem.

Sample Input 1: computer

Sample Output 1: CoMpUtEr

Sample Input 2: programming

Sample Output 2: PrOgRaMmInG

Sample Input 3: PROGRAMMING

Sample Output 3: PrOgRaMmInG
