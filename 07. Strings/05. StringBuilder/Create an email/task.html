Create an email

We need to create an email name for an employee. For this, we need to concatenate the name, the surname, and at the end, the domain "@work.net" and save it into completeEmail variable.

Use StringBuilder to solve the problem.

Sample Input 1:

Bill
Shates

Sample Output 1:

BillShates@work.net

Sample Input 2:

Elon
Tusk

Sample Output 2:

ElonTusk@work.net
